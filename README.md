## 상세내용

```
인테리어가구 쇼핑몰로 이용자의 제품구매, 질문글 등록, 구매목록확인 등과 
관리자의 제품등록 게시글등록 등을 구현.
```

기간 : 2021.04.09 ~ 2021.05.09  
팀구성 : Java개발자 6인   
사용기술 및 사용 툴 : Java, SQL, JSP, CSS&HTML, JavaScript, JQuery,   
                      Apache Tomcet, Oracle, Eclipce(4.16.0)
